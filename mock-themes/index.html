프론트엔드 시안 테마 프리셋

동일한 마크업에 테마 데이터만 바꿔서 다양한 무드의 UI를 빠르게 비교합니다.

Component

카드 패턴

카드의 배경, 보더, 섀도우, 라운드 값이 테마 데이터에 의해 결정됩니다.

Code

코드 블록

fetchTheme(name) {
  return fetch(`/data/${name}.json`).then((res) => res.json());
}
List

체크리스트

  • 색상, 폰트, 레이아웃 토큰
  • Hero / 카드 / 버튼 / 코드
  • Mock JSON → 향후 API 교체
Guidelines

테마는 mock-themes/data의 JSON 스키마만 유지하면 되며, 추후 API 연동 시에도 동일한 키를 반환하도록 맞추면 됩니다.
