API 콘솔을 이용해 샘플 워크플로 실행하기

ActionBlock 구성과 요청/응답 예시를 포함한 문서 상세 페이지 시안입니다.

사전 준비

  • 테넌트별 API Key 발급
  • Webhook URL (Slack/Discord) 등록
  • 액션 실행 권한 확인
POST /api/plugins/execute
Authorization: Bearer {token}
{
  "plugin": "api-console",
  "payload": {
    "method": "POST",
    "url": "https://api.guidebook.wiki/v1/demo",
    "body": { "preview": true }
  }
}
