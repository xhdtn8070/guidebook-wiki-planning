OAuth

카카오 전용 전체 페이지 로그인

TopBar → 로그인 페이지 → Kakao OAuth 콜백 흐름을 프로토타입으로 시연합니다.

  • 카카오 로그인만 제공
  • redirect 파라미터 적용
  • 다크/라이트 모드 전환 연동

로그인

카카오로 5초 만에 시작

이메일/비밀번호 입력 없이 카카오 OAuth만 제공합니다.

OAuth 완료 시 `/auth/callback?provider=kakao&redirect=...`로 이동하는 흐름을 표현합니다.
